
# @gratico/microkernel

@gratico/microkernel
=====

Typescript Interfaces for microkernel specs for use of sdk developers.

Install and use
---------------

To use run `npm install -g @gratico/microkernel`

    import {IKernel, IKernelExtension} from "@gratico/microkernel"
